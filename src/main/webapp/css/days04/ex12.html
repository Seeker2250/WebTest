css 변환을 사용하면 element를 이동, 크기 변경, 회전, 뒤집기 등의 변환을 할 수 있다
    css 2d변환
    transform 속성 == 변환
     : 이동 translate, 크기 scale, 회전 rotate(양수 : 시계방향, 음수 : 반시계방향), 뒤집기 skew(양수 : 시계방향, 음수 : 반시계방향)
     기울기 skewX, skewY
    
     약식 : matrix(scaleX, skewY, skewX, scaleY, translateX, translateY)

    transform-origin 속성 == 변환 원점
    
    css 3d변환
    transform-style 속성 == 변환 스타일
     : 3d 변환 스타일 적용 여부 지정
    perspective 속성 == 원근감
     : 3d 변환 원근감 적용 여부 지정
